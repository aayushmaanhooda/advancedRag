# Advanced RAG

A comprehensive exploration of advanced Retrieval-Augmented Generation (RAG) techniques and implementations.

## Overview

This repository contains experimental implementations and explorations of various advanced RAG approaches, including BM25-based retrieval, hybrid search strategies, and transformer-based models.

## Contents

- **main.ipynb** - Main implementation and experimentation notebook
- **bm25.ipynb** - BM25 retrieval algorithm exploration
- **hybridsearch.ipynb** - Hybrid search implementation combining multiple retrieval strategies
- **hybridnaik.ipynb** - Alternative hybrid search approach
- **theory.txt** - Theoretical background and concepts
- **transformer.pdf** - Transformer model reference documentation
